================================================================================
CACHYOS LINUX — MANUAL SETUP REFERENCE
================================================================================
Companion to the setup_paru.sh / setup_yay.sh installers. The scripts handle
bulk package installs from packages/*.txt; this file documents the manual
steps, service enables, edits, and configuration that the scripts do not do.

CachyOS is Arch-based, so most steps match the Arch reference. The notable
differences are called out with a "CACHYOS:" marker:
  - paru (and yay) ship pre-installed — no AUR helper bootstrap needed.
  - The CachyOS and Chaotic-AUR binary repos are enabled out of the box.
  - GPU drivers are handled by `chwd` (CachyOS Hardware Detection).
  - Mirrors are ranked with `cachyos-rate-mirrors`, not reflector.
  - The default kernel is `linux-cachyos` (manage it with cachyos-kernel-manager).

Order below roughly matches the order to run them on a fresh install.
================================================================================


--------------------------------------------------------------------------------
 1. NETWORK — PPPoE (Daffodilnet)
--------------------------------------------------------------------------------
sudo pacman -S rp-pppoe

nmcli con edit type pppoe con-name "err"
set pppoe.username user@daffodilnet.com
save
yes
quit


--------------------------------------------------------------------------------
 2. AUR HELPER — paru / yay
--------------------------------------------------------------------------------
# CACHYOS: paru is already installed. To confirm:
paru --version

# If you ever need to (re)install, it is a plain repo package here — no AUR
# build required (unlike stock Arch):
sudo pacman -S --needed paru
sudo pacman -S --needed yay


--------------------------------------------------------------------------------
 3. MIRRORS + SYSTEM UPDATE + CORE UTILITIES
--------------------------------------------------------------------------------
# CACHYOS: rank the CachyOS + Arch mirrors with the bundled tool.
sudo cachyos-rate-mirrors

sudo pacman -Syyu
sudo pacman -S --needed base-devel
sudo pacman -S cachyos-keyring archlinux-keyring
sudo pacman -S nano vim git neofetch fastfetch htop bpytop bat curl wget unzip \
    zip xz unrar tar p7zip rsync ntfs-3g exfat-utils bzip2 cmake gcc make clang \
    cargo go nodejs samba bleachbit nvtop cpufetch jdk-openjdk intel-ucode


--------------------------------------------------------------------------------
 4. GPU — DRIVERS via chwd (CachyOS Hardware Detection)
--------------------------------------------------------------------------------
# CACHYOS: do NOT use nvidia-inst. List detected profiles, then auto-install:
chwd -l
sudo chwd -a            # auto-install best driver for detected hardware

# NVIDIA extras (CUDA / container toolkit) if needed:
sudo pacman -S nvidia-open-dkms nvidia-utils nvidia-settings cuda \
    nvidia-container-toolkit libva-nvidia-driver


--------------------------------------------------------------------------------
 5. KERNEL — linux-cachyos
--------------------------------------------------------------------------------
# CACHYOS: the optimized kernel is preinstalled. Manage / switch variants with:
sudo pacman -S cachyos-kernel-manager
cachyos-kernel-manager          # GUI: install linux-cachyos-bore, -lto, -rt, etc.

# Headers must match the running kernel (needed for DKMS modules):
sudo pacman -S linux-cachyos-headers


--------------------------------------------------------------------------------
 6. BLUETOOTH
--------------------------------------------------------------------------------
sudo pacman -S --needed bluez bluez-utils
sudo systemctl enable --now bluetooth


--------------------------------------------------------------------------------
 7. FIREWALL — UFW
--------------------------------------------------------------------------------
sudo pacman -S ufw
sudo systemctl enable --now ufw


--------------------------------------------------------------------------------
 8. SYSTEM OPTIMIZATION
--------------------------------------------------------------------------------
# CACHYOS: cachyos-settings already applies sysctl/udev/zram tuning. Check it:
pacman -Qi cachyos-settings

# Preload — caches frequently used apps (from CachyOS/Chaotic repo, no AUR build)
sudo pacman -S preload
sudo systemctl enable --now preload

# Touch gestures
paru -S touchegg
sudo systemctl enable --now touchegg.service

# CPU frequency / thermal
sudo pacman -S cpupower lm_sensors thermald stress
sudo cpupower frequency-set -g performance
sudo systemctl enable --now cpupower.service
sudo systemctl enable --now thermald

# auto-cpufreq (laptop battery tuning)
paru -S auto-cpufreq
sudo auto-cpufreq --install
